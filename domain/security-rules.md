# 数据安全边界规则

> 版本：2.0.0 | 维护：安全/合规负责人
> 本文件定义 AI 在所有 Phase 中必须遵守的数据安全边界。

---

## 1. 允许输入 AI 的内容

| 类型 | 说明 | 示例 |
|------|------|------|
| 脱敏业务流程 | 用 {merchant_id} 替换真实ID | "merchant {merchant_id} creates subscription..." |
| 示例数据 | 明确标注为 example/test 的虚构数据 | "test card: 4111111111111111" |
| 非生产错误信息 | staging/dev 环境的错误日志（无PII）| "PAYMENT_FAILED: invalid_card in staging" |
| 字段结构 | 脱敏后的字段定义、数据模型 | "amount DECIMAL(20,6), currency VARCHAR(3)" |
| 公开技术文档 | Stripe/Adyen/Visa 公开API文档 | API参考链接 |
| 内部规范文档 | 不含敏感字段的设计规范 | 架构设计文档（已脱敏）|

---

## 2. 禁止输入 AI 的内容

| 类型 | 示例 | 处理方式 |
|------|------|---------|
| 支付卡号 | 4111-1234-5678-9012 | ⛔ 拒绝，要求替换为 {card_number} |
| CVV/CVC | 123 | ⛔ 拒绝 |
| 完整证件号 | 310101199001011234 | ⛔ 拒绝，要求脱敏 |
| API Key | sk_live_xxxx | ⛔ 拒绝，说明不得输入生产凭证 |
| Webhook Secret | whsec_xxxx | ⛔ 拒绝 |
| 数据库密码 | root:password@prod-db | ⛔ 拒绝 |
| 生产DB连接串 | jdbc:mysql://prod.db:3306 | ⛔ 拒绝 |
| 真实交易流水 | txn_123456: ¥99.00, card 4111...9012 | ⛔ 拒绝，要求脱敏 |
| 持卡人PII | 张三, 18612345678, zhangsan@gmail.com | ⛔ 拒绝 |
| 未脱敏生产日志 | 含真实商户ID/金额/卡号的日志 | ⛔ 拒绝 |

---

## 3. 高风险操作清单（必须人工确认）

以下 10 类场景 AI 输出后必须暂停，等待责任人确认后才能执行：

1. **真实支付扣款、退款、拒付处理方案** → 主审：财务/风控
2. **商户资金、平台自有资金、账务分录** → 主审：财务/CFO
3. **生产数据库变更脚本**（ALTER/DROP/TRUNCATE）→ 主审：DBA + 架构
4. **商户权限变更、API Key 重置、Webhook Secret 轮换** → 主审：安全
5. **生产日志、持卡人信息、合同信息分析** → 禁止输入，直接拒绝
6. **风控规则、黑白名单、拒付阈值变更** → 主审：风控负责人
7. **自动任务批量处理**（批量取消、批量退款）→ 主审：产品 + 运营
8. **不可逆状态变更**（Subscription Canceled, Invoice Voided）→ 主审：产品
9. **外部通道、银行、卡组织接口对接方案** → 主审：架构 + 合规
10. **上线回滚困难或影响存量商户的配置变更** → 主审：版本负责人

## 4. 脱敏规范

| 原始数据类型 | 脱敏方式 | 示例 |
|------------|---------|------|
| 商户ID | {merchant_id} 或 M_XXXX | M_1234 |
| 卡号 | 前6后4 或 {card_number} | 411111****9012 |
| 手机号 | 前3后2 或 {phone} | 186****5678 |
| 邮箱 | 首字符+*** | z***@gmail.com |
| 金额 | 可保留或替换为 {amount} | ¥99.00 或 {amount} |
| 交易ID | txn_{hash} 或 {txn_id} | txn_abc123 |
| 证件号 | 前3后4 | 310***1234 |

---

## 5. AI 输出安全声明

每次生成涉及以下内容的方案时，AI 必须在输出末尾附加声明：

```
⚠️ 安全声明：
本输出包含 [支付方案/账务分录/权限变更/数据库变更] 相关内容。
根据团队高风险拦截规则，本内容在执行前必须经过 [责任人] 的人工专项 Review。
本 AI 输出仅为 Draft，不代表最终执行方案。
```
